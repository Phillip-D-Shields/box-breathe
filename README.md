A simple, calming Progressive Web App (PWA) to help reduce anxiety using the 4-4-4-4 box breathing technique. Built to be offline-capable and mobile-friendly.

## The Stack

* **Styling:** Tailwind CSS 4 + DaisyUI 5
* **Hosting:** Cloudflare Pages
* **Features:** Offline PWA support, installable on iOS/Android, haptic visuals.
